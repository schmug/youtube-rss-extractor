# Privacy Policy

**YouTube RSS Extractor** is committed to protecting your privacy. This policy explains what data the extension accesses and how it's used.

## Summary

- We do **not** collect any personal data
- We do **not** track your browsing activity
- We do **not** send data to any external servers
- All processing happens locally in your browser

## Data Collection

**None.** This extension does not collect, store, or transmit any user data.

## Permissions

The extension requests minimal permissions:

| Permission | Purpose |
|------------|---------|
| `activeTab` | Temporary access to the current YouTube page when you click the extension. |
| `scripting` | Allows the extension to read the YouTube page source code to extract Channel IDs and RSS metadata. |

## Network Requests

The extension makes **one** type of network request:

- **YouTube RSS Feed** (`youtube.com/feeds/videos.xml`) â€” Fetched only when you open the extension popup, to display recent videos. This request goes directly to YouTube, not through any third-party server.

## Local Storage

This extension does **not** use local storage, cookies, or any persistent data storage.

## Third-Party Services

This extension does **not** integrate with or send data to any third-party services, analytics, or tracking platforms.
